# Arcane Client Website

Обновленный сайт Arcane Client с новыми функциями.

## Что добавлено

### 1. Кнопка "Покупка" в навигации
- Добавлена кнопка "Покупка" в верхнее меню
- При клике открывается модальное окно с товарами
- Работает как на десктопе, так и на мобильных устройствах

### 2. Модальное окно с товарами
Модальное окно содержит 3 товара:
- **Arcane Client** - подписка на 365 дней
- **Arcane Beta** - навсегда (помечен как "Популярное")
- **Arcane Enjoyner** - подписка на 30 дней

Каждая карточка товара включает:
- Иконку
- Название
- Длительность подписки
- Список функций
- Кнопку "Купить"

### 3. Декоративные синие разводы
По всему сайту добавлены анимированные синие разводы:
- На главной секции
- В секции "Динамический Остров"
- В секции музыки
- В секции модов
- В других секциях
- В футере

Разводы:
- Плавно анимируются
- Не мешают читаемости текста
- Адаптированы для мобильных устройств

## Файлы

- `Rockstar 2.0.html` - основной HTML файл
- `modal.css` - стили для модального окна
- `decorations.css` - стили для декоративных элементов
- `header.css` - стили для шапки сайта
- `index.css` - основные стили сайта

## Использование

1. Откройте файл `Rockstar 2.0.html` в браузере
2. Нажмите на кнопку "Покупка" в верхнем меню
3. Выберите нужный товар в модальном окне
4. Модальное окно закрывается:
   - По клику на крестик
   - По клику вне окна
   - По нажатию клавиши ESC

## Адаптивность

Сайт полностью адаптирован для:
- Десктопов (1920px и выше)
- Планшетов (768px - 1024px)
- Мобильных устройств (до 768px)

## Стиль

Все новые элементы выполнены в стиле сайта:
- Темная тема с прозрачностью
- Эффект размытия (backdrop-filter)
- Градиентные границы
- Плавные анимации
- Синие акценты
